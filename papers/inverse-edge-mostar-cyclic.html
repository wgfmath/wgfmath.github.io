Abstract

This paper settles an inverse problem proposed by Alex and Gutman for the edge Mostar index. They asked whether, for every fixed cyclomatic number c >= 3, all sufficiently large positive integers occur as edge Mostar indices of c-cyclic graphs. We prove that the answer is yes, and we do so in the stronger and chemically natural class of molecular graphs, namely simple connected graphs of maximum degree at most four.

For each fixed c >= 3, we construct explicit one-parameter families of c-cyclic molecular graphs whose edge Mostar indices realize all sufficiently large integers. The construction uses a long cycle as a backbone, two short chords, and a controlled number of two-edge ears placed in two separated regions. A complete edge-distance classification is carried out: for every parity case of the long cycle, each edge is assigned its exact edge-Mostar contribution. As a result, the edge Mostar index becomes an affine function of the cycle length in each parity class, and the two parity classes together cover all sufficiently large integers.

Thus Conjecture 4.2 of Alex and Gutman is confirmed. The result shows that, once the cyclomatic number is fixed at any value c >= 3, the edge Mostar index has no asymptotic obstruction in the molecular-graph setting: apart from finitely many small exceptions, every positive integer is attainable.
